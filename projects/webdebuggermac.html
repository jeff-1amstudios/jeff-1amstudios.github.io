WebDebugger for Mac is a native OSX proxy useful for debugging at the HTTP level. It is written in ObjC/C++, using Cocoa and Boost libraries.

  • HTTP
  • HTTPS tunneling
  • HTTPS interception (provides a dummy SSL cert for the browser->proxy connection so plain-text traffic can be viewed)
  • The application that opened the connection is discovered, so all requests can be filtered by application
  • Connection timing statistcs
  • Fake DNS entries, DNS cache
  • Animated powered collapsible panels
  • 100% native code and native Cocoa interface
Status: Work in progress - no release candidate available
